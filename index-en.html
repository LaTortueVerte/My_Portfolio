During my course and on my personal time I realized several projects


Terraphore

The winning student project of the edition MasterProject 2023 organized within our school, Terraphore is a rover permacole hedgerow planter. It can move autonomously, dig holes and plant several species of hedges.

Within this project, I worked on the integration of the Ardupilot autopilot and the Pixhawk 8C board, the motor drivers, the connection between all the rover modules, the wiring, the tests and finally the video.

link to the video

Draw.smthg

Draw.smthg is a personal project, it was a drawing website. Each day, a new theme was presented, the user could then make a unique drawing on the proposed subject. He could afterwards visualize the drawings of other users. The drawing tool is in pure javascript, and the back-end part is in PHP and a SQL part for user account management.

link to github repository

IOT - RGB LED pannel

IOT RGB Pannel is also a personal project, it is the interfacing of a 32x16 LED RGB screen of the adafruit brand. I found that LED panels were too complicated to use because of the wiring, and programming, so I decided to create a PCB to facilitate its use with an ESP8266.

link to github repository
